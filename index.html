CSS Basics

This page demonstrates the fundamental concepts of CSS (Cascading Style Sheets) and how they work together to create visually appealing web pages. CSS allows us to separate content from presentation, making our HTML more semantic and our designs more maintainable.

Key CSS concepts covered include selectors, properties, values, and the cascade. We're using various types of selectors including element selectors, class selectors, and pseudo-selectors to target specific HTML elements.

The CSS Box Model

The CSS Box Model is fundamental to understanding layout in CSS. Every element on a web page is considered a rectangular box, consisting of content, padding, border, and margin.

Box Model Examples

Small Box (10px padding, 10px margin, 1px border)
Default Box (20px padding, 20px margin, 3px border)
Large Box (40px padding, 30px margin, 8px border)

Notice how the padding creates space inside each box, the margin creates space around each box, and the border provides a visual boundary. The total width and height of an element includes all these components.

Layout Techniques

Modern CSS provides several powerful layout techniques. Below we demonstrate flexbox for creating responsive card layouts with consistent spacing using the box model properties.

Margin

Creates space outside the element's border. Margins collapse vertically between adjacent elements, and can be negative values to pull elements closer together.

Padding

Creates space inside the element, between the content and the border. Padding is always positive and affects the element's total size unless box-sizing is set to border-box.

Border

The border sits between margin and padding. It can have different styles, colors, and widths, and contributes to the element's total dimensions.

Visual Styling
